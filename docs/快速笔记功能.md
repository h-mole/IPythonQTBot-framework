# 📝 快速笔记 - 快捷键和右键菜单速查卡

## ⌨️ 快捷键列表

### 文件操作
| 快捷键 | 功能 | 图标 |
|--------|------|------|
| `Ctrl+S` | 💾 保存笔记 | 💾 |

### 查找替换
| 快捷键 | 功能 | 图标 |
|--------|------|------|
| `Ctrl+F` | 🔍 查找 | 🔍 |
| `Ctrl+H` | 🔄 替换 | 🔄 |

### 编辑操作
| 快捷键 | 功能 | 图标 |
|--------|------|------|
| `Ctrl+Z` | ↶ 撤销 | ↶ |
| `Ctrl+Y` | ↷ 重做 | ↷ |
| `Ctrl+X` | ✂ 剪切 | ✂ |
| `Ctrl+C` | 📋 复制 | 📋 |
| `Ctrl+V` | 📌 粘贴 | 📌 |
| `Ctrl+A` | ☑ 全选 | ☑ |

## 🖱️ 右键菜单

在编辑器任意位置**点击鼠标右键**，即可打开菜单：

```
┌─────────────────────────────┐
│ ↶ 撤销         (Ctrl+Z)    │
│ ↷ 重做         (Ctrl+Y)    │
├─────────────────────────────┤
│ ✂ 剪切         (Ctrl+X)    │
│ 📋 复制         (Ctrl+C)    │
│ 📌 粘贴         (Ctrl+V)    │
├─────────────────────────────┤
│ ☑ 全选          (Ctrl+A)    │
├─────────────────────────────┤
│ 🔍 查找         (Ctrl+F)    │
│ 🔄 替换         (Ctrl+H)    │
├─────────────────────────────┤
│ 💾 保存         (Ctrl+S)    │
└─────────────────────────────┘
```

---

## 🎯 常用场景

### 快速保存
```
编辑中 → Ctrl+S → ✓ 已保存
```

### 查找内容
```
1. Ctrl+F
2. 输入关键词
3. 回车查找下一个
```

### 批量替换
```
1. Ctrl+H
2. 输入查找内容和替换内容
3. 点击"全部替换"
```

### 使用右键菜单
```
右键点击编辑器 → 选择功能 → 完成操作
```

---

## 💡 智能提示

### 菜单项状态
- **灰色**: 当前不可用（如未选中时剪切为灰色）
- **黑色**: 当前可用

### 自动检测
- 有选中文本时才会启用剪切/复制
- 有撤销历史时才会启用撤销
- 剪贴板有内容时才会启用粘贴

---

## 🚀 效率技巧

### 纯键盘工作流
```
编辑 → Ctrl+S 保存 → Ctrl+F 查找 → Ctrl+H 替换
```

### 鼠标 + 键盘组合
```
右键 → 选择功能 → 按快捷键
```

### 推荐工作流
1. **编辑时**: 随时 `Ctrl+S` 保存
2. **修改时**: 用 `Ctrl+Z/Y` 撤销/重做
3. **查找时**: `Ctrl+F` 快速定位
4. **批量修改**: `Ctrl+H` 全部替换

## ⚠️ 注意事项

### Linter 误报
如果看到以下警告，可以安全忽略：
- `Qt.CustomContextMenu` 未知
- `QKeySequence.Undo` 未知

这些是正常的 PySide6 API。

### 平台差异
- **Windows/Linux**: 使用 Ctrl 键
- **macOS**: 自动切换为 Command 键
